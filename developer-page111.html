FOR SOFTWARE DEVELOPERS

If you want to develop analytic software then the Monitor+ JSON API is the ideal solution. As well as being easy to use, Monitor+ is designed to be the most powerful reference data API currently available and can be used with virtually any programming language.

Monitor+ handles user authentication and session tokens as well as user favorite datasources, datasets and metadata. It supports requests for multiple datasets, for any date range, with custom fills and averages, selectable spark line data and more. It can return results in a variety of JSON formats and has been created to make it easier for developers to create great products - fast.

We take care of the complete back-end so developers only need Monitor+ to access the range of back-end features. This ability greatly speeds up product development time, allowing developers to focus on creating great front end analytic solutions.

Monitor+ takes care of session tokens and and user authentication management. There are no databases to connect to and no SQL to write - Monitor+ does it all.

Monitor+ supports time series data and controls individual user access down to datasource or data category level. We also support individual user 'favorite' lists that contain a list of datasets that are chosen and managed by the user.




To get started, first take a look at the Monitor+ API product page and then the Monitor+ API documentation page to understand the API features, its implementation and to become familiar with the list of available functions and their parameters.
