FOR SOFTWARE DEVELOPERS

If you need to develop analytic software then the Monitor+ JSON API is the ideal solution. As well as being easy to use, Monitor+ is designed to be the most powerful reference data API currently available and can be used with virtually any programming language.

Monitor+ supports requests for multiple datasets, date ranges, custom fills, custom averages, spark line data and more. It can return results in a variety of JSON formats and makes it easier for developers to create great products - fast.

Monitor+ provides developers with all the data access features that they need. There are no databases to connect to and no SQL to write - Monitor+ does it all. Having no back-end to deal with speeds up product development time and allows developers to focus on creating great front end solutions.

Monitor+ takes care of user authentication and controls individual user access down to datasource/data category level. It uses session tokens and supports user favorite datasources, datasets and metadata as well as user 'favorite' lists containing lists of datasets that are chosen and managed by each user.




To get started, we suggest you first take a look at the Monitor+ API product page to get an overview of the system and then the Monitor+ API documentation page to discover the list of available API functions and their parameters.
